# Đóng góp cho ClickFlow

Tạo branch theo `feat/`, `fix/`, `chore/`, `docs/`; giữ thay đổi nhỏ, có acceptance criteria và kiểm thử phù hợp. Commit theo Conventional Commits, ví dụ `feat: add task status`. Pull request phải dùng template, không kèm secret, và chỉ merge sau review. Quy tắc chi tiết xem [coding standards](docs/engineering/CODING_STANDARDS.md).
