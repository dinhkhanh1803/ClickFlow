# Definition of Done

Task hoàn thành khi đáp ứng acceptance criteria, được review, không còn TypeScript/lint error, test liên quan pass, không có console error nghiêm trọng, có loading/empty/error state và validation. Logging, docs, migration, security review và deployability phải phù hợp thay đổi; không được làm hỏng tính năng cũ.
