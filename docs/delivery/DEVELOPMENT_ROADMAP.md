# Development Roadmap

0. Product Foundation: cấu trúc, docs, standards, templates.
1. Frontend Foundation: Next.js, design system, shell, routing, mock data, state, auth UI.
2. Frontend Features: dashboard, projects/tasks, views, time, reports, settings.
3. Backend Foundation: NestJS, config, Prisma/PostgreSQL, auth, error/logging, Swagger.
4. Backend Business Modules: project, task, time, attachment, comment, activity, report.
5. Integration: API client, auth, fetching, optimistic update, upload.
6. Testing/Hardening: automation, security, performance, accessibility, bug fixes.
7. Deployment: Vercel, Render, database, environment, domain, monitoring, backup.
