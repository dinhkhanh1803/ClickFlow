# ADR-001: Monorepo

**Status:** Accepted. **Context:** Web/API cần contracts và config chung. **Decision:** pnpm workspace + Turborepo. **Alternatives Considered:** multi-repo, npm workspaces. **Consequences:** đồng bộ tốt nhưng cần boundary kỷ luật. **Risks:** coupling. **Review Date:** trước Phase 1.
