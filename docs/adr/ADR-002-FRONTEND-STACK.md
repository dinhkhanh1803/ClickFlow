# ADR-002: Next.js frontend

**Status:** Accepted. **Context:** cần web UI hiện đại. **Decision:** Next.js, TypeScript, Tailwind, shadcn/ui, TanStack Query, Zustand, React Hook Form, Zod, dnd-kit, Recharts. **Alternatives Considered:** SPA thuần, Vue. **Consequences:** hệ sinh thái phù hợp Vercel. **Risks:** giới hạn server/client boundary. **Review Date:** trước Phase 1.
