# Tài liệu ClickFlow

- [`api/`](./api/README.md): OpenAPI artifact và hướng dẫn API theo feature/task.
- `product/`: định hướng, phạm vi và quy tắc nghiệp vụ.
- `engineering/`: kiến trúc và tiêu chuẩn kỹ thuật.
- `delivery/`: roadmap, backlog, release và rủi ro.
- `runbooks/`: quy trình vận hành, incident và backup/restore.
- `adr/`: quyết định kiến trúc.
