# ClickFlow Web

Next.js App Router frontend foundation for ClickFlow. This phase intentionally uses mock data and local mock authentication only.

## Commands

- `pnpm dev` — starts the local app.
- `pnpm lint` — runs ESLint.
- `pnpm typecheck` — runs TypeScript validation.
- `pnpm test` — runs Vitest unit tests.
- `pnpm build` — creates the production build.
- `pnpm exec playwright test` — runs the browser smoke test.

## Phase 1 scope

Light-first application shell, dashboard mock, navigation route shells, login/reset-password mock flows, design tokens and quality automation. No backend calls or persisted data are present.
